# Alpha-Release v0.2.0

## Changelog
- Fehler beim Öffnen der Einstellungen behoben
- Plugin Parser behoben
  - Strings aus Sub-Cells, z.B. in Whiterun werden nun vollständig extrahiert
  - PERK EPF2 und PERK EPFD zum Parser hinzugefügt (betrifft v.a. CACO)
- mögliche Crash-Ursache beim Modlist-Scan behoben
  - bitte rückmelden, ob Crashes ohne Fehler immer noch auftreten
- DownloadListDialog hinzugefügt (s. Screenshot)
  - neueste Übersetzung und neueste File werden vorausgewählt
  - Übersetzung und Main File können frei gewählt werden, wenn mehrere verfügbar
- Zusammenführen von Quest- und Dialog-Strings behoben (Danke an @skyhorizon_ !)
- Importieren und Installieren von Übersetzungen drastisch optimiert (Danke an @skyhorizon_ !)
- Neue *Suchen und Ersetzen* Funktion zum Editor hinzugefügt

## Hinweise
- **Aufgrund der Veränderungen am Parser, muss die gesamte Database per Strg+A > Rechtsklick > *Delete Selected* neu aufgesetzt werden!!**

## Bekannte Probleme
- Fehler beim Aufheben der NXM-Verknüpfung (MO2 muss nach Beenden des ATs evtl. neu verknüpft werden)

# Alpha-Release v0.2.6

## Changelog
- MESG ITXT behoben, es sollte jetzt nicht mehr zu Mismatches (z.B. "Ja" für "No") kommen
  - betroffene Übersetzungen müssen neu installiert werden
- API Setup lässt sich nun über die Einstellungen neustarten
- API Setup an Design des restlichen UIs angepasst
- deutsche Übersetzung für UI hinzugefügt (Dokumentation noch immer ausschließlich auf Englisch)
- es werden im Output nur noch JSON-Dateien erstellt, wenn es Strings gibt, die reinmüssen
- nach dem Import einer lokalen Übersetzung wird nun die Modliste aktualisiert, um betroffene Plugins entsprechend grün zu färben
- Strg+Enter Shortcut zum Translator Dialog hinzugefügt, um Strings schnell abzuspeichern
- Hinweise in Dokumentation und zum Tooltip des "Build DSD Output" Buttons hinzugefügt, um deutlich zu machen, wann ein neuer Output notwendig ist
